# Novelty Decision

Decision: full-scale deterministic submission candidate.

The strongest contribution is a protocol and benchmark boundary: model disagreement should identify the physical probe that would distinguish candidate embodied hypotheses, while accounting for observation signatures, safety gates, physical cost, aborts, false abstention, and abstention behavior. V2 hardening remains as the negative control, and the v3 benchmark scales the evidence to 414,720 compact condition rows.

The paper is suitable as a deterministic benchmark/reporting submission. It should not be framed as hardware validation or a proof that physical probes always dominate latent disagreement.
